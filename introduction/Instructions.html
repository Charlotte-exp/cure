  2. The total number of rounds is randomly determined. After each round, there is a {{ delta }}% probability that the interaction with the same person will continue for at least another round. This is as if we would roll a four sided dice: if a 1 or a 2 comes up, the game ends after this round, but if a 3 or 4 comes up, the game continues for at least another round. So, for instance, if you are in round 2, the probability there will be a another round is {{ delta }}% and if you are in round 9, the probability there will be another round is also {{ delta }}%.
